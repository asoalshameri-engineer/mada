# mada
مدى يقدم عبارات تفاعلية وتحفيزية لمساعدة المستخدم على البقاء ايجابيا

## الوصف
مدى يقدم عبارات تفاعلية وتشجيعية لمساعدة المستخدم على البقاء متحمسًا وإيجابيًا على هيئة تحديات للحفاظ على صحته النفسية.  
يتكون المشروع من ثلاثة فصول، كل فصل يحتوي على 7 مواضيع، وكل موضوع يحتوي على عبارات تحفيزية وتفاعلية.


## الفصول
1. الفصل الأول: [ختَر وانطلق]  
2. الفصل الثاني: [صحوة: عندما تفتح الحياة أبوابها من جديد]  
3. الفصل الثالث: [فن العيش الواعي: من التأمل إلى الحرية]  
    



## المميزات
- عبارات تحفيزية يومية  
- تفاعل مع العبارات (تم الانجاز/مشاركة/ ملاحظات)  
- تصميم بسيط وجذاب  
- سريع وسهل الاستخدام
- تفعيل الوضع الليلي
- قائمة تحتوى على الاعدادات وتحتزى على كلا من:[الملف الشخصي / الإعدادات الشخصية / معلومات التطبيق / الأسئلة الشائعة / تسجيل الخروج] 



## طريقة الاستخدام
1. افتح الملف index.html في المتصفح.  
2. اختر الفصل الذي تريد تصفحه.  
3. استمتع بالعبارات التفاعلية والتحفيزية.  



## التقنيات المستخدمة
- HTML  
- CSS  
- JavaScript
